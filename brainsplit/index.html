About this game

During my second year of studies at INSA Toulouse, I followed an optional web programming course. It covered the basics of HTML5, CSS3, Javascript and Jquery. To put into pratice what I had just learned, I tried and developped this simple game.

It has a very simple design and the code is probably outdated. But I was quite happy with it at the time, and I still find it funny to play. It is actually very challenging (at least for me) after 60 seconds, when all the four minigames are running !
